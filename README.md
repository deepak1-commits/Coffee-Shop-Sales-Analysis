# ☕ Coffee Shop Sales Analysis

An end-to-end Data Analytics project focused on analyzing coffee shop sales data using **SQL** and **Power BI**. The project transforms raw transactional data into interactive dashboards that help identify sales trends, customer behavior, product performance, and store-level insights for data-driven business decisions.

---

## 📌 Project Objective

The objective of this project is to analyze coffee shop sales data and uncover actionable insights by answering key business questions related to:

- Sales performance
- Customer purchasing patterns
- Store performance
- Product category analysis
- Peak sales hours
- Monthly sales trends

---

## 🛠️ Tools & Technologies

- **Power Query** – Data cleaning and transformation
- **DAX** – KPI calculations and custom measures
- **Excel/CSV** – Dataset source

---

## 📂 Dataset

The dataset contains transactional sales records with attributes such as:

- Transaction ID
- Transaction Date & Time
- Store Location
- Product Category
- Product Type
- Unit Price
- Quantity Sold
- Total Bill

---

## 📊 Dashboard KPIs

The dashboard tracks the following key performance indicators:

- 💰 Total Sales
- 🧾 Total Orders
- 📦 Total Quantity Sold
- 📈 Month-over-Month Growth
- 📅 Daily Sales Trend
- 🏪 Store-wise Sales
- ☕ Product Category Performance
- ⭐ Top 5 Best-Selling Products
- ⏰ Sales by Hour & Day

---

## 📈 Key Business Insights

- Identified monthly and daily sales trends.
- Analyzed peak business hours to understand customer traffic.
- Compared sales performance across multiple store locations.
- Evaluated product category contribution to overall revenue.
- Identified Top 5 revenue-generating products.
- Analyzed weekday vs weekend sales patterns.
- Created interactive filters for dynamic business reporting.
